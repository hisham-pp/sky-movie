# Governance

Sky Movie is maintained by the project maintainers and shaped by community
contributions.

## Maintainer Responsibilities

Maintainers are responsible for:

- Reviewing issues and pull requests.
- Keeping the project direction coherent.
- Protecting project quality, security, and community health.
- Publishing releases when appropriate.
- Updating documentation as the project changes.

## Decision Making

Most decisions are made through issue and pull request discussion. Maintainers
aim to explain decisions clearly, especially when declining a proposed change.

Project direction should prioritize:

- A useful and reliable experience for users.
- Maintainable implementation choices.
- Clear documentation and setup.
- Respectful community participation.

## Becoming a Maintainer

Regular contributors may be invited to take on maintainer responsibilities
after demonstrating helpful judgment, consistent participation, and care for
the project's goals.

## Project Changes

Large changes should start with an issue or proposal before implementation.
This helps contributors align on the problem, scope, and trade-offs before
significant work begins.
